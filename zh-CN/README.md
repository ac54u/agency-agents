# 🎭 代理中文版 (73个)

The Agency 项目的中文翻译版本，涵盖以下部门：

- **工程开发 (engineering)** — 前端、后端、移动、AI、DevOps、数据库等
- **设计 (design)** — UI/UX、品牌、视觉设计
- **安全 (security)** — 安全架构、渗透测试、威胁检测
- **测试 (testing)** — QA、性能、无障碍审计
- **产品 (product)** — 产品经理、反馈分析、A/B实验
- **项目管理 (project-management)** — PM、Jira、会议记录
- **App Store优化** — ASO 排名优化

所有文件格式兼容 OpenCode，可直接使用。也可作为其他 AI 工具的 agent 提示词参考。

## 使用方式

将需要的 agent 文件复制到对应工具的 agents 目录即可。

```
# OpenCode
cp zh-CN/*.md ~/.config/opencode/agents/

# Claude Code
cp zh-CN/*.md ~/.claude/agents/
```
